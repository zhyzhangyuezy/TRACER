# Outputs Directory

This directory stores released result summaries and auxiliary artifacts used in
the paper snapshot.

Released subdirectories:

- `results/`: stored JSON summaries, prediction exports, and audit inputs used
  to regenerate the reported tables and figures
- `expert_evidence_annotation_packet/`: blinded annotation scaffold for future
  human evaluation of retrieved analogs

Local runtime logs and temporary probes are intentionally excluded from the
GitHub release package.
