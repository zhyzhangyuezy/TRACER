# TRACER archival release v1.0.0

This release provides the public archival package for the TRACER paper,
including:

- model code
- processed benchmark bundles
- stored result summaries and audit exports
- figure-generation scripts

Notes:

- The main repository is kept within standard GitHub file-size constraints.
- The oversized AIT-ADS canonical-events table is distributed as a separate
  release asset: `ait_ads_canonical_events.zip`.
- Manuscript source files and compiled PDFs are intentionally kept outside the
  public code-and-data repository.
- Third-party raw-source mirrors are not redistributed in the git-tracked
  repository; the release instead provides processed bundles, scripts, and
  provenance notes required for reproduction of the paper artifacts.
